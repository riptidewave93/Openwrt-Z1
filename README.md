# Openwrt-MR18

Bringup repo for the Cisco Meraki Z1 on OpenWRT!

Building
-----
#### Build Only
`./build.sh`

#### Modify Configs and Build
`./build.sh modify`

Note that you will need to run a modify on the first compile to select the NAND image in the OpenWRT menuconfig.
